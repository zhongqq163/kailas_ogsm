# 技术栈

更新时间：2026-06-22

## 技术选择

- 前端：原生 HTML + CSS + JavaScript。
- 后端：Cloudflare Pages Functions，当前新增 `/api/state` 读写接口。
- 数据库：本地兜底使用浏览器 `localStorage`；云端共享使用 Cloudflare D1。
- 部署：Cloudflare Pages，静态输出目录为 `web/`，D1 binding 为 `DB`。
- 其他：初始数据来自 `IT数据应用支持组_钟启庆_OGSM_2026V2.0.xlsx`，网页支持 JSON 导入导出。

## 运行命令

```bash
直接打开 web/index.html
```

Cloudflare 本地预览：

```bash
npx wrangler pages dev web --compatibility-date=2026-06-25 --d1=DB=agsm-dashboard-db
```

## 关键目录

- `web/index.html`：页面结构。
- `web/styles.css`：页面样式。
- `web/app.js`：初始数据、交互逻辑、本地保存、导入导出。
- `functions/api/state.js`：Cloudflare Pages 数据接口。
- `schema.sql`：Cloudflare D1 表结构。
- `wrangler.toml`：Cloudflare 部署配置。
- `DEPLOY_CLOUDFLARE.md`：Cloudflare 部署步骤。
